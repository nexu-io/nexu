**1/3**  
Nexu v0.1.6 is here 🚀

Introducing **ClawHub** — a built-in skill marketplace with **10,000+ skills** from the OpenClaw ecosystem.

🧩 **ClawHub highlights:**  
→ 10,000+ OpenClaw skills, searchable and categorized  
→ Batch install with queue management — pick 10 skills, install them all at once  
→ One-click sync with the latest community skills

No more hunting on GitHub. Your agent just got a superpower store.

**2/3**  
Nexu v0.1.6 is here 🚀

This release is all about **speed, stability, and staying alive in the background**.

⚡ **3x faster cold start** — from ~22s down to ~7s. Reopen from background in just ~200ms.  
🛡️ **Crash-proof architecture** — Controller and OpenClaw now run as independent system services. If one crashes, it auto-recovers. No app restart needed.  
🔌 **Background mode** — Close the window, your bots keep running. Reopen from Dock instantly. No state lost.  
🧩 **ClawHub** (formerly Explore) — A brand-new skill marketplace with batch install + queue management. Finding the right skill just got way easier.  
🔥 **Dev hot reload** — Edit controller code, see changes in ~3s. No restart, no waiting.  
🎨 **Polished launch experience** — New animated splash screen. Status shows "Starting up → Connecting → Running" instead of flashing "Offline".  
🐛 Fixes: WeChat error messages now say "Reconnect required" instead of generic "Error". Display sleep no longer blocked while app is active.
