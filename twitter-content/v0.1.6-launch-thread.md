**2/3**  
Nexu v0.1.6 — full changelog 🚀

1. 🧩 **ClawHub** (formerly Explore) — A brand-new skill marketplace with batch install + queue management. Finding the right skill just got way easier.
2. 🛡️ **Crash-proof architecture** — Controller and OpenClaw now run as independent system services. If one crashes, it auto-recovers. No app restart needed.
3. 🔥 **Dev hot reload** — Edit controller code, see changes in ~3s. No restart, no waiting.
4. ⚡ **3x faster cold start** — from ~22s down to ~7s. Reopen from background in just ~200ms.
5. 🔌 **Background mode** — Close the window, your bots keep running. Reopen from Dock instantly. No state lost.
6. 🎨 **Polished launch experience** — New animated splash screen with real-time status.

🐛 Fixes: WeChat error messages now say "Reconnect required" instead of generic "Error". Display sleep no longer blocked.

Full changelog → github.com/nexu-io/nexu/releases/tag/v0.1.6
