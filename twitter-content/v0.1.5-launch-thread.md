## v0.1.5 更新要点

### 新功能（5 项）
| 功能 | 用户价值 |
|---|---|
| **Skill 导入弹窗**（zip 上传 + 中英双语） | 用户可以直接拖入 zip 导入 Skill，不需要命令行操作 |
| **聊天气泡 Markdown 渲染** | 对话里的列表、链接、代码块都能正常显示，阅读体验大幅提升 |
| **动态端口分配** | 不再因为端口冲突导致启动失败 |
| **关于页面 + 应用内更新检查** | 用户可以在客户端里直接检查新版本 |
| **工作空间与运行时分析事件** | 后续可追踪产品使用数据，帮助改进体验 |

### Bug 修复（关键 3 项）
| 修复 | 用户价值 |
|---|---|
| **飞书重启死循环** | 之前飞书渠道会反复重启不可用，现在彻底解决 |
| **退出时僵尸进程** | 关掉 Nexu 后不再残留进程、不再阻碍重新安装 |
| **冷启动配置重载** | 每次重启不再重新加载所有配置，启动更快更稳 |

### 新贡献者
- @NickHood1984、@zoeforfun 首次贡献
