# 30-Minute Quick Start

Get your first AI Agent running in under 30 minutes.

## Prerequisites

- A Mac with nexu installed ([Download](https://nexu.io/download))
- An account on at least one chat platform: Feishu, Slack, or Discord

## Step 1: Launch nexu (2 min)

1. Download and install the nexu desktop app.
2. Double-click to open — no configuration files needed.
3. Sign in or create a nexu account.

## Step 2: Connect a channel (10 min)

Pick one channel and follow the setup guide:

- [Feishu](/guide/channels/feishu) — only need App ID + App Secret
- [Slack](/guide/channels/slack) — one-click app creation via manifest
- [Discord](/guide/channels/discord) — need Application ID + Bot Token

After filling in your credentials, click "Connect" in the nexu client.

## Step 3: Configure a model (5 min)

1. Go to **Settings** → **Models** in the nexu client.
2. Choose a model provider (or use nexu Official for instant access).
3. If using BYOK (Bring Your Own Key), paste your API key and verify the connection.

See [Model Configuration](/guide/models) for detailed provider setup.

## Step 4: Install a skill (5 min)

1. Go to **Skills** in the nexu client.
2. Browse the skill catalog and pick one that fits your use case.
3. Click "Install" — the skill loads automatically.

See [Skill Installation](/guide/skills) for more details.

## Step 5: Chat with your Agent (3 min)

1. Back on the Home page, you'll see your channel showing "Connected".
2. Click "Chat" to jump to your chat platform.
3. Send a message to your bot — it should reply using your configured model and skills.

That's it! Your AI Agent is live. 🎉
