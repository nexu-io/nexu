# 一分钟快速上手

下载 nexu，连接渠道，你的 AI Agent 即刻就绪。

## 第一步：启动 nexu

1. [下载](https://nexu.io/download)并安装 nexu 桌面客户端。
2. 双击打开，登录或注册账号。

## 第二步：连接渠道

选择一个渠道，按照指南完成接入：

- [飞书](/zh/guide/channels/feishu) — 只需 App ID + App Secret
- [Slack](/zh/guide/channels/slack) — 一键创建预配置应用
- [Discord](/zh/guide/channels/discord) — 需要 Application ID + Bot Token

填入凭证，点击「Connect」。

## 第三步：开始对话

渠道显示「Connected」后，点击「Chat」跳转到聊天平台，给机器人发一条消息即可。

搞定！你的 AI Agent 已上线 🎉
