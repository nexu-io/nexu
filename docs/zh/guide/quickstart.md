# 30 分钟快速上手

30 分钟内让你的第一个 AI Agent 跑起来。

## 前置条件

- 一台安装了 nexu 的 Mac（[下载地址](https://nexu.io/download)）
- 至少一个聊天平台的账号：飞书、Slack 或 Discord

## 第一步：启动 nexu（2 分钟）

1. 下载并安装 nexu 桌面客户端。
2. 双击打开——无需配置文件。
3. 登录或注册 nexu 账号。

## 第二步：连接渠道（10 分钟）

选择一个渠道，按照指南完成接入：

- [飞书](/zh/guide/channels/feishu) — 只需 App ID + App Secret
- [Slack](/zh/guide/channels/slack) — 一键创建预配置应用
- [Discord](/zh/guide/channels/discord) — 需要 Application ID + Bot Token

填入凭证后，在 nexu 客户端点击「Connect」。

## 第三步：配置模型（5 分钟）

1. 在 nexu 客户端进入 **Settings** → **Models**。
2. 选择模型提供商（也可以直接使用 nexu 官方模型，开箱即用）。
3. 如果使用自带密钥（BYOK），粘贴 API Key 并验证连接。

详见 [模型配置](/zh/guide/models)。

## 第四步：安装技能（5 分钟）

1. 在 nexu 客户端进入 **Skills**。
2. 浏览技能目录，选择适合你场景的技能。
3. 点击「Install」——技能会自动加载。

详见 [技能安装](/zh/guide/skills)。

## 第五步：和 Agent 对话（3 分钟）

1. 回到首页，你的渠道会显示「Connected」。
2. 点击「Chat」跳转到聊天平台。
3. 给机器人发一条消息——它会使用你配置的模型和技能进行回复。

搞定！你的 AI Agent 已上线 🎉
